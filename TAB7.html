This page focuses on the introduction of string classes through text interpretation and code examples, including the basic methods of creating strings, merging strings, obtaining string length and string format, and further explains the application of stringBuffer and stringBuilder in modifying strings. Finally, a large number of API function constructions and methods related to string classes are provided.

Creat String


The simplest way to create a string is as follows:

String greeting = "rookie tutorial";

When a String constant is encountered in the code, the value here is "rookie tutorial" and the compiler will use this value to create a string object.

Like other objects, you can use keywords and construction methods to create String objects. The String class has 11 construction methods that provide different parameters to initialize the string, such as providing a character array parameter:

Connection string

The String class provides a way to connect two strings:

string1.concat(string2);

Returns the new string that string2 connects string1. You can also use the concat () method on string constants, such as:

"My name is". concat("Runoob ");

More commonly, the'+'operator is used to connect strings, such as:

"Hello," + " runoob" + "!"

The results are as follows: "Hello, runoob!"

Create a formatted string

We know that printf () and format () methods can be used to output formatted numbers.

The String class returns a String object instead of a PrintStream object using the static method format ().

The static method format () of the String class can be used to create reusable formatted strings, not just for one-time printout.

String Buffer


When modifying strings, StringBuffer and StringBuilder classes are required.

Unlike String classes, StringBuffer and StringBuilder classes can be modified many times without generating new unused objects.

StringBuilder class was proposed in Java 5. The biggest difference between StringBuilder and StringBuffer is that StringBuilder's method is not thread safe (cannot be accessed synchronously).

Because StringBuilder has a speed advantage over StringBuffer, StringBuilder class is recommended in most cases. However, when the application requires thread safety, the StringBuffer class must be used.

Date and Sample Date Format

This page shows a very important time module in java learning. The website introduces in detail how to obtain the current time, how to use calendar class to obtain specific years, months, dates, weeks, etc. through video tutorials and corresponding text explanations and code examples. It also gives a simple method of formatting dates-SampleDate. Finally, it gives some formatting and coding information of date corresponding characters.

Creat Date

// Initialize Date object

Date date = new Date();

//Use toString () function to display date and time

System.out.println(date.toString());

Creat DateFormat

SimpleDateFormat ft = new SimpleDateFormat ("yyyy-MM-dd hh:mm:ss");

This line of code establishes the format of conversion, where yyyy is the complete year ad, MM is the month, dd is the date, HH:mm:ss is the hour, minute and second.

Note: some formats are uppercase and some are lowercase, for example, MM is the month and mm is the minute; Hh is a 24-hour system, while HH is a 12-hour system.

The compilation results of the above examples are as follows: The current time is: 2018-09-06 10:16:34
