JAVA basic grammer

A Java program can be thought of as a collection of objects, and these objects work together by calling each other's methods. The following briefly introduces the concepts of classes, objects, methods, and instance variables.

1.Object: An object is an instance of a class, with state and behavior. For example, a dog is an object, and its states are: color, name, breed; behaviors are: shaking tail, barking, eating, etc.
2.Class: A class is a template that describes the behavior and state of a class of objects. 3.Method: Method is behavior, a class can have many methods. Logical operations, data modification, and all actions are completed in the method.
4.Instance variables: Each object has a unique instance variable, and the state of the object is determined by the value of these instance variables.

basic grammer


When writing Java programs, you should pay attention to the following points:

1.Case sensitivity: Java is case sensitive, which means that the identifier Hello is different from hello.
2.Class name: For all classes, the first letter of the class name should be capitalized. If the class name.consists of several words, the first letter of each word should be capitalized, for example MyFirstJavaClass.
3.Method name: All method names should start with a lowercase letter. If the method name contains several words, the first letter of each subsequent word is capitalized.
4.Source file name: The source file name must be the same as the class name. When saving a file, you should use the class name as the file name (remember that Java is case sensitive), and the file name suffix is .java. (If the file name and the class name are different, it will cause a compilation error).
5.Main method entry: All Java programs begin execution with the public static void main (String [] args) method.

Java modifier


Like other languages, Java can use modifiers to modify methods and properties in the class. There are two main types of modifiers:

1.Access control modifiers: default, public, protected, private.
2.Non-access control modifiers: final, abstract, static, synchronized.
We will discuss Java modifiers in depth in later chapters.

Java variables


There are several types of variables in Java

1.Local variables
2.Class variables (static variables)
3.Member variables (non-static variables)

Java array


Arrays are objects stored on the heap and can hold multiple variables of the same type. In later chapters, we will learn how to declare, construct, and initialize an array.
