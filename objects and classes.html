objects and classes

Object: An object is an instance of a class (the object is not to find a girlfriend), and has state and behavior. For example, a dog is an object, and its states are: color, name, breed; behaviors are: shaking tail, barking, eating, etc.

Class: A class is a template that describes the behavior and state of a class of objects.

Create object


Objects are created based on classes. In Java, use the keyword new to create a new object. The following three steps are required to create an object:

1.Declaration: Declare an object, including object name and object type.
2.Instantiation: Use the keyword new to create an object.
3.Initialization: When creating an object using new, the constructor is called to initialize the object.
