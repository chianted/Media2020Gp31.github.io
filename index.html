There are three characteristics of object-oriented
1. Encapsulation
Hide the properties and implementation details of the object, only provide public access, isolate the changes, make it easy to use, and improve the reusability and security.
2. Inheritance
Improve code reusability; inheritance is the premise of polymorphism.
3. Polymorphism
A reference variable defined by a parent class or interface can point to an instance object of a subclass or concrete implementation class. It improves the expansibility of the program.

Advantages of polymorphism
1. Eliminate coupling between types
2. Replaceability
3. Expandability
4. Interface
5. Flexibilityv 6. Simplicity
